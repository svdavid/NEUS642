NEUS 642 Week 6: Current source density from multi-channel recordings in cortex.

To get started:
* install the pycurl package in your python environment. This package allows you to download data and files from remote servers using the http protocol.
  * `conda install pycurl`
  
* Make sure you have the latest version of the course materials:
  * `cd <my_NEUS_git_path>`
  * `git pull`

* Open [week6_CSD_students.ipynb](week6_CSD_students.ipynb) in Jupyter.
