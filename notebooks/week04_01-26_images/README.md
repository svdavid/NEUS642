NEUS 642 Week 4: Image Processing for Microscopy

To get started:
* Make sure you have the latest version of the course materials:
  * `cd <my_NEUS_git_path>`
  * `git pull`
* install the pyables and opencv packages after activating your python environment. You'll need pytables for class and opencv for homework.
  * `conda install pytables opencv`
* Open `notebooks/week04_01-26_images/Week4_image_processing_homework_base.ipynb' in Jupyter and complete the exercises.
